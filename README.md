# E-Commerce Website

A full-stack E-Commerce web application built using modern web technologies. This platform allows users to browse products, add items to the cart, place orders, and manage their shopping experience through an intuitive interface.

---

## 🚀 Features

- User Registration & Login
- Product Listing
- Product Details Page
- Add to Cart
- Remove from Cart
- Shopping Cart Management
- Order Placement
- Responsive Design
- Secure Authentication
- Database Integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- CSS / Bootstrap

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Tools
- Git
- GitHub
- MongoDB Atlas
- Vite

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/e-commerce-website.git
```

### Backend Setup

```bash
cd backend

npm install

npm start
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_CONNECTION_STRING

JWT_SECRET=YOUR_SECRET_KEY
```
